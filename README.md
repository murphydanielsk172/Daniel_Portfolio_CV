# Portfolio Website

A minimal overview of the portfolio built with React, Vite, and Tailwind CSS.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

## Build

Create a production build:

```bash
npm run build
```
